# 取件助手 (Logistics PWA)

一个简单、高效、隐私友好的物流短信解析与取件管理工具。

## 🌟 特性

- **智能解析**: 自动从物流短信中提取取件码、快递公司和存放地点。
- **PWA 支持**: 支持安装到 iOS 和 Android 桌面，像原生应用一样使用。
- **离线优先**: 核心功能支持离线访问，无需网络即可查看已存包裹。
- **本地存储**: 数据完全存储在手机本地的 IndexedDB 中，保护隐私且速度极快。
- **响应式设计**: 适配各种移动端屏幕尺寸，简洁直观。

## 🚀 快速开始

1. **访问应用**: 打开部署后的 GitHub Pages 链接。
2. **安装应用**:
   - **iOS**: 点击分享按钮 -> “添加到主屏幕”。
   - **Android/Chrome**: 点击地址栏旁边的“安装”按钮或响应安装提示。
3. **添加包裹**:
   - 点击“添加新包裹”按钮。
   - 粘贴物流短信（例如：“【菜鸟驿站】您的圆通快递包裹已到...请凭123456取件”）。
   - 点击解析并确认存入。

## 🛠️ 技术栈

- **HTML5 / CSS3**: 使用 Flexbox 实现响应式布局。
- **JavaScript (Vanilla)**: 原生逻辑处理，不依赖外部框架。
- **IndexedDB**: 用于浏览器端持久化数据存储。
- **Service Worker**: 实现资源缓存与离线访问。
- **Manifest.json**: PWA 标准配置文件。

## 📄 部署

本项目支持一键部署到 GitHub Pages。只需将代码推送到 GitHub 仓库并开启 Pages 服务即可。
